#  NLP Sentiment Analysis Dashboard using Machine Learning

An end-to-end **Natural Language Processing (NLP)** and **Machine Learning** project that analyzes movie reviews and predicts whether the sentiment is **Positive** or **Negative**.

This project demonstrates the complete Data Science workflow, from data preprocessing and model training in **Jupyter Notebook** to deploying a professional **Flask Web Application** with an interactive user interface.

---

##  Live Demo

🔗 **Live Website:** https://nlp-sentiment-analysis-kp9w.onrender.com

---

#  Project Overview

The project applies **Natural Language Processing (NLP)** techniques to preprocess movie reviews and uses a **Multinomial Naive Bayes** classifier with **TF-IDF Vectorization** to perform sentiment classification.

The trained model is integrated into a Flask web application that predicts sentiment in real time through a modern and responsive interface.

#  Installation

Clone the repository

```bash
git clone https://github.com/your-username/NLP-Sentiment-Analysis.git
```

Move to the project directory

```bash
cd NLP-Sentiment-Analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Train the model

```bash
python train_model.py
```

Run the Flask application

```bash
python app.py
```

Open the browser

```
http://127.0.0.1:5000
```
